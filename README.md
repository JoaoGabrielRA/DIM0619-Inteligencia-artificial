# DIM0619-Inteligencia-artificial

Repositório para armazenar o código da avaliação 3 da disciplina de inteligencia artificial - DIM0619

- Este trabalho apresenta a implementação e avaliação de um modelo de regressão logística multiclasse aplicado ao problema de reconhecimento de dígitos manuscritos utilizando o dataset MNIST. O modelo foi treinado do zero, utilizando aprendizado supervisionado, a partir de imagens em tons de cinza de dígitos de 0 a 9. O pipeline incluiu etapas de carregamento dos dados, visualização, normalização, treinamento, avaliação quantitativa e testes com imagens externas.
